ğŸ”§ How IA-HASH works (minimal protocol)
1. Normalize text
prompt_normalized = bytes(prompt, "utf-8")
response_normalized = bytes(response, "utf-8")

2. Hash both parts
h_prompt = SHA256(prompt_normalized)
h_response = SHA256(response_normalized)

3. Compute the combined hash
h_total = SHA256(
    version + "|" +
    prompt_id + "|" +
    h_prompt + "|" +
    h_response + "|" +
    model + "|" +
    timestamp
)

4. Sign h_total using the issuerâ€™s private key (Ed25519 recommended)
signature = sign(private_key, h_total)

5. Bundle everything into an IA-HASH Document

This document can later be verified without accessing the original issuer.
